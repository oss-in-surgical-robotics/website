# oss-in-surgical-robotics.github.io/website

## Local preview of the site
```bash
# Install once (requires Node.js)
npm install -g serve
# Run from the project folder
cd /path/to/hsmr-workshop
serve .
```
Open web-browser for http://localhost:3000  


## Clone repo
```bash
git clone git@github.com:oss-in-surgical-robotics/website.git
```
